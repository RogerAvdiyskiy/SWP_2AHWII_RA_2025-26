# Lieblingsessen App - Distribution Package

## Files Included
- main.ts - Server application (Hono framework)
- db.ts - Database logic (SQLite)
- lieblingsessen.db - Sample data database
- deno.json - Deno configuration file
- urls.rest - REST API testing file
- front-end/ - Web interface files
- start-app.bat - Windows launcher
- README.md - Complete user guide

## Quick Setup
1. Extract zip file
2. Double-click start-app.bat (Windows)
3. Open http://localhost:8000 in browser

## Requirements
- Deno 1.36+ installed
- No additional dependencies needed

## Support
- All platforms supported (Windows/macOS/Linux)
- Local-only application
- No internet required after setup
